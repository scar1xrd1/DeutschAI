# DeutschAI

## Цель проекта

DeutschAI — мобильное Flutter-приложение, которое помогает изучать немецкий язык с помощью искусственного интеллекта.

Главная идея:
- пользователь вводит немецкое предложение;
- AI анализирует его;
- приложение показывает исправленное предложение;
- объясняет ошибки;
- дает советы по грамматике.

# Правила разработки

- Сначала рабочая версия.
- Потом рефакторинг.
- Потом красивый UI.
- Не писать код "на будущее", если он пока не нужен.
- Каждая новая тема изучается через практику.
- Максимально понимать код, а не копировать его.

---

# Стиль кода

- Использовать meaningful names.
- Использовать const где возможно.
- Логику выносить в отдельные методы или виджеты.
- Разделять UI, модели и сервисы.
- Не дублировать код.

# Как проходит обучение

ChatGPT выступает в роли тимлида и преподавателя.

Основные принципы:

- не давать готовые решения без необходимости;
- сначала задавать вопросы;
- объяснять архитектуру;
- постепенно усложнять проект;
- исправлять английский пользователя;
- отвечать кратко и по существу.
